# Mandolin Scale Visualizer

A static single page web app that displays a mandolin fretboard with interactive scale visualization.

## Features

- **Circle of Fifths Key Selector**: Click on any key in the circle of fifths to select it
- **Scale Selection**: Choose from Major, Minor, Pentatonic, or Blues scales
- **Orientation**: Toggle between horizontal and vertical fretboard display
- **Note Names**: Toggle visibility of note names on the fretboard
- **Visual Distinction**: Root notes are displayed in black with white text; non-root scale notes are displayed in white with black text (inverse)

## Tech Stack

- TypeScript 6 (typescript@next)
- Preact
- D3.js
- Vite

## Getting Started

### Install dependencies

```bash
npm install
```

### Run development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

### Preview production build

```bash
npm run preview
```
